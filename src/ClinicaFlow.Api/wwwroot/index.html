Benvenuto su ClinicaFlow

Interfaccia web per la gestione operativa di una clinica privata. Da qui puoi accedere alle tre aree del progetto: Back Office, Medico e Paziente.

Il frontend è stato progettato per consumare le API REST del backend ASP.NET Core già presenti nel repository e mostrare in modo chiaro pazienti, medici, disponibilità, appuntamenti e referti.
